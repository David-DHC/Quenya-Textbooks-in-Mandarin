            I. 冠词

            Quenya没有不定冠词。

            任何情况下，Quenya定冠词都是i。

            名词的数

            Quenya名词分为单数、双数、复数、集合复数。

            1. 双数：成双成对出现的事物，可以使用双数。

              注释：不是所有的“两个”事物，都能被认为是“一对”、“一双”。请注意区分。

              变形方法：

              • 以辅音字母结尾的名词，加"-u"。若该辅音字母前为长元音，则缩短之。

              • 以短元音、双元音结尾的名词，加"-t"。特殊情况如下：

                • 以d/t+短元音结尾的词，去掉这个短元音，而后加"-u"。
                • 表示身体部位的词，若以短元音、双元音结尾，则去掉最后一个元音字母，再加"-u"。同时也可采用正常的"-t"后缀。

              • 以长元音结尾的词，缩短长元音后加"-u"。

              例词：

              • ranco "arm" + -u = rancu "a pair of arms" 双臂。
              • pé "lip" + -u = peu "a pair of lips" 双唇。
              • má "hand" + -t = mát "a pair of hands" 双手。
              • Marto "Tower" + -u = Martu "Two Towers" 双塔。

            2. 集合复数：指成群出现的事物，略宽泛于英语“集合名词”的概念。

              注释：目前，研究人员对Quenya是否存在集合复数仍有争议，这里我们姑且将其列出。

              变形方法：

              • 以辅音字母结尾的词，加"-li"。
              • 以元音字母结尾的词，加"-i"。

              例词：

              • elen "star" + -i = eleni "stars" 群星。
              • alda "tree" + -li = aldali "woods" 森林。
              • nírë "tear" + -li = níreli "tears" 泪雨。

            3. 复数：定义：非单个、非成对、非成群的事物，均使用复数。

              注释：依照此定义，即使某事物是两个，只要非成对，也要用复数（而非双数）。

              变形方法：

              • 以辅音字母结尾的名词，加"-i"。
              • 以元音字母结尾的名词，加"-r"。特殊情况如下：
              • 以-ë结尾的名词（以-ië或-llë结尾的除外），去掉"-ë"加"-i"。

              例词：

              • ráma "wing" + -r = rámar "wings" 翅膀。
              • tië "road" + -r = tier "roads" 道路。
              • lassë "leaf" + -i = lassi "leaves" 树叶。
              • aran "king" + -i = arani "kings" 国王们。
