              I. 其他的常见前缀与后缀

              • en- "re-, again" 重新。
              • ava-/nec- "without" 没有...的。
              • ar(a)- "near, next to" 在...旁边的。
              • nan- "backwards" 向后地。
              • ú- "in-, un-, mal-" （表示反义）。
              • e- （连接人称代词后缀，表示强调）。
              • -ië "-ness" （接在形容词后面构成名词）。

               

              II. 动词equë的用法

              • 含义：equë一词，意为“说”，无单复数变化，无时态变化。许添加一个人称代词后缀作为主语。

                equë一词仅限用于“equë + 说话者+所说的话”这一个句型。在任何其他句型中，均应使用quet。

              • 例句

                • "Utúlien," Elendil quentë. "Elendil said: 'I've come.'" 伊兰迪尔说：“我来了”。
                • Equë Elendil: "Utúlien." "Elendil said: 'I've come.'" 伊兰迪尔说：“我来了”。
                • Equen: "Utúlien." "I said: 'I've come.'" 我说：“我来了。”

              III. 动词auta的过去式与完成式

              • 含义：物理上的“离开”；抽象层面的“消失、逝去”。
              • 在第一种含义下，该词的过去式为oantë，完成式为oantië。
              • 在第二种含义下，该词的过去式为vánë，完成式为avánië。

               

              IV. 以-o结尾的名词的复数

              一般变为-ui。例词：

              • urco "bogey" + -i = urqui "bogeys"。
              • Orco "orc" + -i = orqui "orcs"。
              • rusco "fox" + -i = rusqui "foxes"。
              • ango "snake" + -i = angwi "snakes"。
              • lango "throat" + -i = langwi "throats"。

              V. 以-u结尾的动词

              • 变形方法：其各种变形均可仿照A类动词的变形方法。特殊情况如下：

                • 不定过去时：加"-i"。
                • 一般将来时：加"-va"。
                • 现在完成时：去掉-u后按照I类动词的变形方法进行变形。
                • 现在分词：加"-la"。若该词的主干元音可以拉长，则拉长。

              • 例词：

                • palu "spread":

                  • palu + -i = palui "spread"。
                  • palu + -va = paluva "will spread"。
                  • palu + -ië = apálië "have spread"。
                  • palu + -la = pálula "spreading"。

                • nurru "murmur":

                  • nurru + -i = nurrui "murmur"。
                  • nurru + -va = nurruva "will murmur"。
                  • nurru + -ië = unurrië "have murmurred"。
                  • nurru + -la = nurrula "murmurring"。
